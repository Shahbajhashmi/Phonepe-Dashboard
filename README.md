# 📊 PhonePe Transaction Insights Dashboard

A modern and interactive **PhonePe Transaction Insights Dashboard** built using **Microsoft Power BI** and **Microsoft Excel**. This project provides a comprehensive analysis of transaction performance, user behavior, payment status, and service-wise insights through visually appealing dashboards.

---

## 📌 Project Overview

The dashboard is designed to help users monitor and analyze PhonePe transaction data efficiently. It offers interactive visualizations and key performance indicators (KPIs) for better business insights and decision-making.

---

## 🚀 Features

- 📈 Total Transactions Overview
- 💰 Total Transaction Value
- 👥 Unique Users Analysis
- ✅ Success Rate Monitoring
- 📅 Monthly Transaction Trends
- 🎯 Age Segment Contribution
- 🛍️ Service-wise Transaction Analysis
- 🏆 Top 5 Users by Transaction Value
- 📊 Weekday vs Weekend Usage Analysis
- 🎛️ Interactive Filters (Month & Payment Status)
